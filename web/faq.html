Enquiries. Questions heard at the counter

Is Vault100 really free?

Yes — free and MIT-licensed open source, with no accounts, no subscriptions and no ads. Run it, study it, modify it, self-host it, forever.

Do my files ever leave my device?

Never. All cryptography runs locally in a dedicated Worker thread of your browser — or fully offline in the CLI and desktop app. Disconnect the network and the bureau keeps working exactly the same.

Is the web counter as strong as the desktop engine?

Yes. The web counter uses the same primitives through WASM libsodium and WebCrypto, stamps identical parameters, and opens CLI vaults byte-exactly (and vice versa). Each page proves it on arrival by decrypting a reference vault made by the desktop build — watch the inspection stamp, upper right.

What if I forget a vault's combination?

No one on earth can recover it — not even the bureau, which keeps no ledger of combinations, keys or escrow copies. Write passphrases down; store keyfiles like house keys, with a copy laid safely away.

Which algorithms protect my vaults?

XChaCha20-Poly1305 authenticated streaming encryption, an optional AES-256-GCM cascade (double-lock), Argon2id memory-hard key derivation with 108 graded notches, and optional 256-bit keyfiles as a physical second factor. Parameters are stamped in each vault's public header.

What exactly is a .v100key keyfile?

A small file holding 256 random bits, minted by the keyfile press (Annex D) or the CLI. When demanded at sealing, every later opening needs BOTH the combination and the file — a physical second factor, VeraCrypt-style.

Does the bureau see or log anything about me?

No accounts, no cookies, no analytics, no telemetry, no uploads. The only things kept are three desk preferences inside your own browser's storage (ink, shift, sweep seconds). The privacy writ enumerates everything.

Can I use Vault100 offline or self-host it?

Yes. After the first visit the web counter works with the network cable severed (offline PWA). Self-hosting is a zero-dependency Node server or the provided Dockerfile; the CLI and desktop app never need a network at all.

Can I change a vault's passphrase later?

Yes — Form 100-R (or python -m vault100 passwd) re-seals only the vault's key wrap; the body is never re-encrypted, so a 10 GB vault's combination changes as fast as a 10 KB one.

May my business use Vault100?

Yes. MIT means commercial use, modification and redistribution are all permitted — just keep the license notice. No permission letter needed; the counter serves institutions and citizens alike.
