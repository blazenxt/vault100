Take-home. Five ways to keep the bureau

way the first · lightest

Carry the counter (PWA)

Visit the bureau once, then install it from your browser's install / add to home screen prompt. Thereafter it opens with the network cable severed — the whole counter is cached on your premises.

if the button sleeps, this browser wants its own menu: Install app / Add to Home Screen 
# or, the manual way — browser menu → "Install app"
# thereafter: cable severed, counter still serving
way the second · the engine

The command engine

The reference implementation — every build must match it byte-for-byte.

git clone https://github.com/blazenxt/vault100.git
cd vault100
pip install -r requirements.txt
python -m vault100 encrypt secret.pdf --cascade
python -m vault100 decrypt secret.pdf.v100
python -m vault100 bench          # the timekeeper
python -m vault100 verify x.v100  # custody check, nothing written
python -m vault100 share split combo.txt -n 5 -m 3  # quorum press
way the third · heavier window

The desktop box

A Tkinter counter in the night-ledger livery — same vaults, same notches, live strength bar, stopwatch bench, recently-handled forms, and a verify desk that proves vaults without writing a byte.

# Debian/Ubuntu first: sudo apt install python3-tk
python -m vault100.gui
way the fourth · your own seal

Run your own bureau

Serve the counter yourself — a zero-dependency Node host or one Dockerfile, hardened headers included.

docker build -t vault100 . && docker run -p 8080:8080 vault100
# — or, nothing to install beyond Node ≥ 20 —
node web/server.mjs   # → http://localhost:8080
way the fifth · in the handset

The pocket annex (Android)

A real, signed .apk — the whole bureau folded into the handset. It asks for zero permissions — not even internet — and serves every form from inside itself, so the counter keeps working with the cable severed, forever. Same worker, same vaults, byte-for-byte as the reference engine. It takes shares from any app, opens .v100 vaults straight from the file manager, and bars screenshots & screen recordings of the ledger.
