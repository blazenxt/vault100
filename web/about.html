Charter copy. The bureau, its creed, its credentials

why the bureau exists

Vault100 was opened in 2026 by BlazeNXT on one stubborn belief: honest encryption software should never promise magic. No tool on earth can vow “impossible to decrypt” — so instead the bureau layers independent defenses, publishes the entire ledger of its works, and lets you keep every key. If one layer ever falls, the next still holds.

the creed, stamped thrice

Zero knowledge. Files, combinations and keys never leave your device. The server only hands out printed paperwork.
Open stock. MIT-licensed from the first ledger line — audit it, fork it, print it, sell it. The notice travels along.
No ledger of you. No accounts, no telemetry, no escrow, no analytics. The bureau keeps zero copies of anything.

three bodies, one cloth

  • The web counter — this very bureau, an offline-capable PWA. Sealed or opened vaults never cross the wire.
  • The command engine — python -m vault100, the reference implementation every build must match byte-for-byte.
  • The desktop box — a Tkinter counter for those who prefer a heavier window, same vaults, same notches.
